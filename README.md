## 使用方法
```sh
# 克隆代码
git clone https://github.com/bitzh-2022-2023-Web-Lab/lastProject.git

# 安装包管理器(可选)
npm i pnpm -g

# 进入文件夹
cd lastProject

# 下载依赖
pnpm install

# 调试
pnpm run dev

# 打包
pnpm run build
```
